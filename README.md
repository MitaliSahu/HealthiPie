# HealthiPie - Smart AI Health Tracker 🥧

HealthiPie is a full-stack web application designed to help users track their wellness journey using AI-driven insights. 

## 🚀 Features
* **AI HealthBot**: Personal wellness consultant powered by Google Gemini API.
* **Smart Dashboard**: Real-time tracking of weight, water intake, BMI, and macros.
* **Goal Oriented**: Specifically optimized for muscle gain and weight management profiles.
* **PWA Enabled**: Installable on mobile and desktop for a native app experience.

## 🛠️ Tech Stack
* **Backend**: Python (Flask)
* **Frontend**: Tailwind CSS, Chart.js
* **AI Engine**: Google Generative AI (Gemini)
* **Deployment**: Render / Gunicorn

## 📦 Installation
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`.
3. Set your `GOOGLE_API_KEY` in environment variables.
4. Run `python app.py`.
